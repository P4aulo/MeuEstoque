# MeuEstoque

Sistema simples de controle de estoque em **C# .NET**.

## Funcionalidades
- Registrar entrada de produtos
- Registrar saída de produtos
- Consultar quantidade disponível em estoque
- Evitar erros manuais no controle

## Tecnologias utilizadas
- .NET 8
- C#
